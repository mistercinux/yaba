Yet Another Backup Assistant:
-----------------------------

Yaba est un simple script écrit en python, dont objectif est de faciliter la sauvegarde de plusieurs emplacements en configurant une liste de fichiers à ignorer. (Puis c'est amusant et ca me fait bosser!)

- Compression des fichiers sauvegardés:
    Les fichiers sauvegardés seront archivés au format .tar.bz2
    (La lib tarfile permet de compresser les éléments directement.)

- Fichier de conf : -> /etc/yaba/yaba.conf.
